Rules

You will be given a random number at the start of the game.

The four ores to the left are each given an unknown randomized value, and clicking them will add that much to your total.

You win the game by matching the number listed above the ores. You lose the game if you go over that total.

Every time the game starts, all values will be randomized.

Good luck and have fun!
